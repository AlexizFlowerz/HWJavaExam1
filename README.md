Программа по скоращению ссылок Фетисова А.
Запускаете в Main.
Начальное меню предлагает либо использовать свой айди, либо сгенерировать.
В люом случае вы будете направлены в свой ЛК, в котором сможете создавать уникальные короткие ссылки и переходить по ним и удалять.
А также ставить лимит по переходам.
